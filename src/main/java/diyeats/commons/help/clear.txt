clear all the data in an interval between two dates

    clear /startdate START_DATE /enddate END_DATE

For example:

    clear /startdate 1/1/2019 /enddate 3/1/2019

will delete all meal items from 1 January 2019 to 3 January 2019 (inclusive)

    clear /startdate 1/1/2019 /enddate 1/1/2019

will delete all meal items on 1/1/2019.
